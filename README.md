Простий голосовий асистент, створений на Python з використанням бібліотек `speech_recognition` та `pyttsx3`.

## Можливості
- Розпізнавання голосових команд
- Відповіді голосом
- Прості команди керування
- (ще в розробці +більше можливостей)

## Як запустити
1. Встановіть: `pip install -r requirements.txt`
2. Запустіть `main.py`
